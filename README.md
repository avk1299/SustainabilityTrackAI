# 🌱 SustainabilityTrackAI

> An AI-powered sustainability dashboard and personal environmental companion that helps users monitor, understand, and reduce their environmental impact.

---

## 📖 Overview

SustainabilityTrackAI is a modern web application designed to help individuals build sustainable habits by tracking daily resource consumption and providing AI-driven insights.

Instead of acting as a one-time carbon footprint calculator, SustainabilityTrackAI continuously assists users in making environmentally responsible decisions through interactive dashboards, personalized recommendations, and habit tracking.

---

## ✨ Features

- 🌍 Carbon footprint monitoring
- ⚡ Electricity usage tracking
- 💧 Water consumption tracking
- 🗑 Waste management monitoring
- 🚗 Transportation impact tracking
- 📊 Interactive sustainability dashboard
- 🤖 AI-powered sustainability assistant
- 💡 Personalized eco-friendly recommendations
- 📈 Progress tracking and analytics
- 🎯 Sustainable habit monitoring
